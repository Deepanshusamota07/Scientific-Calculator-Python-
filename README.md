# Scientific-Calculator-Python
This is about the scientific calculator made in python and the gui is made in html css.
The file named as app.py is the main python code for the scientific calculator and the file named as index.html in the folder templates is the main code for the graphic user interface and the file in the styles folder and named as styles.css includes the design and all thing which is in css .
We are using the flash library in this calculator.
